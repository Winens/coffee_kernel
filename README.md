# Make Your Own Coffee Distro
It is so simple to make.
* 1- Build Kernel `cmake . && make`
* 2- Create root directory `./make_distro --root`
* 3- Start Building Disk Image `./make_distro --confirm`

You Can Run With QEMU `./make_distro --run`
