# Make Your Own Coffee Distro
* 1- Build Kernel `cmake . && make`
* 2- Create root directory `./make_distro.sh --root`
* 3- Start Building Disk Image `./make_distro.sh --confirm`

You Can Run With QEMU `./make_distro.sh --run`
